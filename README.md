# Tinkoff_bot_notifier
The program sends notifications about the status of stock prices and dividends via the telegram bot API (you need to get a key).
The program receives information about promotions using the Tinkoff API (you need to get a key). Requests are made using the curl library.

To create an executable file in Windows, just run makexec.bat.
